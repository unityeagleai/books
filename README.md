# Unity Eagle Creative Books

A collection of experimental interactive creative books made with Claude Fable.

This is a static GitHub Pages site.

Each book may contain its own page-specific styling, scripts, animations, breathing interactions, and experimental layouts.

## Books

- Dust and Static: a Neural Monastery book where passed paragraphs erode into dust and static as the reader moves forward.
- What the Lake Kept: a Neural Monastery book where static text reconstructs where the reader looks, while the ending remains partly unrecovered.
